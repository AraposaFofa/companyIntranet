Como funciona a semantica de versionamento?

Uma semantica de versionamento é feita da seguinte forma:

vX.Y.Z...

X significa a versão major, onde ocorrem mudanças grandes.

Y significa a versão minor, onde ocorrem pequenas, mas importantes mudanças.

Z significa a versão patch, onde ocorrem correções de bugs.

Para saber mais, clique nolink.
